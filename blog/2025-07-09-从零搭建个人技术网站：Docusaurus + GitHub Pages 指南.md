## 为什么选择这个方案？

在尝试了多种静态网站生成器后，我发现 **Docusaurus + GitHub Pages** 组合是最适合技术人员的方案，原因如下：

1. **完全免费**：GitHub Pages 提供免费托管和 HTTPS
2. **技术友好**：原生支持 Markdown、React 和 MDX
3. **开箱即用**：内置文档系统、博客系统、搜索功能
4. **持续部署**：GitHub Actions 自动构建发布
